# Virtual Engagement Analyzer

## Problem Statement
In virtual meetings and online classrooms, presenters cannot accurately determine
whether participants are attentive or engaged. Current platforms mainly provide
attendance data, which does not reflect actual involvement or understanding.

## Solution Overview
This project is an early-stage prototype that calculates a basic engagement score
using interaction metrics such as attendance percentage, chat activity, and reactions.
